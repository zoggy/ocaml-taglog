Quick install with Opam

The easiest way to install is to use OPAM. Install Opam, then, as a regular user, fetch and install an OCaml version and install Taglog. The whole can be done with the following opam commands: 

$ opam init --comp=4.02.1
$ opam update
$ opam install taglog
Dependencies

To build Taglog, you will need:

  • OCaml >= 4.02.1,
  • Ocf >= 0.2.0.
From the sources

Get the sources from git repository:

$ git clone https://github.com/zoggy/ocaml-taglog.git

Or pick one of the releases:

  • 0.1.0 [June 18, 2015]
    • Initial release
Compilation

Just type 

$ ./configure && make && make install

This will install the taglog package with ocamlfind.
